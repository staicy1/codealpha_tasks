\# CodeAlpha Music Player Task



This project is developed as part of the CodeAlpha Frontend Development Internship.



\##  Task Description

Build a Music Player using HTML, CSS, and JavaScript with audio controls and interactive features.



\##  Features

\- Play and Pause functionality

\- Next and Previous track buttons

\- Song title and artist display

\- Progress bar with real-time updates

\- Volume control

\- Responsive user interface



\##  Technologies Used

\- HTML

\- CSS

\- JavaScript



\##  Project Structure

\- index.html

\- style.css

\- script.js

\- assets/ (audio files \& images)
